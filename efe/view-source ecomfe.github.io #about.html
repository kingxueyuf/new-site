<section class="ria">
<h3><a name="ria">RIA</a></h3>
<dl>
<dt><a href="http://github.com/ecomfe/er/" target="_blank">er</a></dt>
<dd>
<p class="outline">Enterprise RIA Framework</p>
<p class="description">富浏览器端 web 应用的框架，适用于并能很方便地构建一个整站式的 AJAX web 应用。ER的目的在于让业务开发工程师可以关注于 数据（Model） 、 业务（Action） 和 视图（View） 这三部分，而不需要关心如数据怎么获取 、 权限如何管理 、 URL 如何控制 等技术性问题。</p>
<a href="http://github.com/ecomfe/er/" target="_blank">GitHub</a>
<a href="http://ecomfe.github.io/api/er/3.1.0-beta.3/index.html" target="_blank">Doc</a>
</dd>
<dt><a href="http://github.com/ecomfe/esui/" target="_blank">esui</a></dt>
<dd>
<p class="outline">Enterprise Simple UI library</p>
<p class="description">一套功能完善，并具有强大扩展机制的 UI 组件库。基于模板、声明式地编程方式让视图更具表达能力。</p>
<a href="http://github.com/ecomfe/esui/" target="_blank">GitHub</a>
<a href="http://ecomfe.github.io/api/esui/3.1.0-beta.3/index.html" target="_blank">Doc</a>
</dd>
<dt><a href="http://ecomfe.github.io/etpl/" target="_blank">etpl</a></dt>
<dd>
<p class="outline">Enterprise Template</p>
<p class="description">灵活、具有强大复用能力的高性能的模板引擎，适用于 WEB 前端应用中视图的生成，特别是 SPA(Single Page APP) 类型的应用。</p>
<a href="http://github.com/ecomfe/etpl/" target="_blank">GitHub</a>
<a href="http://ecomfe.github.io/etpl/" target="_blank">Site</a>
</dd>
</dl>

<h3><a name="interface">Interface</a></h3>
<dl>
<dt><a href="http://ecomfe.github.io/est/" target="_blank">est</a></dt>
<dd>
<p class="outline">EFE Styling Toolkit</p>
<p class="description">est 是一个基于 Less 的样式工具库，帮助您更轻松地书写 Less 代码。est 提供了一系列方便快捷的 mixin，只在调用时才输出代码。est 不希望提供直接给 HTML 调用的类名，用「样式类」污染 HTML 代码的语义。当然您也可以根据自己的项目需求基于 est 搭建样式类库，提供类名接口来进行快速开发。</p>
<a href="http://github.com/ecomfe/est/" target="_blank">GitHub</a>
